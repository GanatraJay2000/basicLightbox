basicLightbox

How to use

Download the files. Download

Add the css and js files to your project

link css

link js

Example

<div class="open-image">
<img class="image" src="image1.png" alt="">
</div>
<div class="open-image">
<img class="image" src="image2.png" alt="">
</div>
<div class="open-image">
<img class="image" src="image3.png" alt="">
</div>

You just have to add any number of images in the manner as above and you can make them display any way you want.
I have used a wrapper outside that is using flexbox.
Don't forget to link the js and css files.
